# K_MACROPAD_6SW_1RTE
6 key macropad with a rotary encoder.

This is a project for hackclub stardance.
# Features
- EC11 Rotary Encoder for volume up and down
- Has 6 Cherry MX Mechanical keys
- Snap_fit_case
